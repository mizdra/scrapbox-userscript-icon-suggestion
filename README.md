# scrapbox-userscript-icon-suggestion

ページ内に埋め込まれているアイコンを suggest して挿入できる UserScript。

## 使い方

[icon-suggestion - Scrapbox カスタマイズコレクション](https://scrapbox.io/customize/icon-suggestion) を参照。
